| 共通部品 | Remote Control | 赤外線リモコン | 数字/制御キー入力を送信 |

## 1. ピン接続表

| Arduino UNO R3 | 接続先 | 用途 |
|:--|:--|:--|
| DCジャック/Vin | 9V1A ACアダプタ | Arduino本体電源 |
| 5V | ブレッドボード+レール | ロジック系/サーボ電源供給 |
| GND | ブレッドボード-レール | ロジック系/サーボGND供給 |
| D2 | IR受信モジュール OUT | IR信号受信 |
| D3 | TM1637 CLK | 7セグクロック |
| D4 | TM1637 DIO | 7セグデータ |
| D8 | パッシブブザー + | 警告音出力 |
| D9 | サーボ SG90 信号線(橙) | 施錠/解錠制御 |
| D10 | 330Ω抵抗(R) -> RGB LED R | 赤LED制御 |
| D11 | 330Ω抵抗(G) -> RGB LED G | 緑LED制御 |
| D12 | 330Ω抵抗(B) -> RGB LED B | 青LED制御 |

> 注意:
> - サーボ電源(赤)は+レール、サーボGND(茶)は-レールへ接続する。
> - サーボをArduino 5Vで駆動するため、負荷が大きい場合は動作不安定になることがある。
> - D0/D1は未使用(Serial予約)。
> - D10-D12はLED専用とし、SPIは未使用前提。

## 4. 実装チェックリスト

- D0/D1に何も接続していない
- D2, D3, D4, D8, D9, D10, D11, D12 の重複割り当てがない
- RGB LEDの各色に330Ω抵抗を1本ずつ入れている
- ジャンパコードの色分けを行い、信号線と電源線を識別できる
- サーボ電源（赤/茶）が+レール/-レールに正しく接続されている
- Arduino本体は9V1A ACアダプタ(DCジャック/Vin)で給電している
- サーボ動作時に電圧降下やリセットが発生しないことを確認している
- 通電前に極性と短絡を最終確認した

---

## 5. 実配線手順（1本ずつ接続）

1. Arduino 5Vをブレッドボード+レールへ、Arduino GNDを-レールへ接続する。
2. IR受信モジュールを VCC/GND/OUT = +レール/-レール/D2 で接続する（GRY配線時は G->GND, R->VCC, Y->OUT）。
3. TM1637を VCC/GND/CLK/DIO = +レール/-レール/D3/D4 で接続する。
4. パッシブブザーを +/− = D8/-レール で接続する。
5. RGB LEDは共通カソードを-レールへ接続し、R/G/Bを330Ω経由でD10/D11/D12へ接続する。
6. サーボは信号線(橙)をD9、電源線(赤)を+レール、GND線(茶)を-レールへ接続する。
7. 9V1A ACアダプタをArduino DCジャック（またはVin）へ接続する。

---

## 6. 通電テストとトラブル切り分け

1. 通電はArduino本体のみから開始し、発熱や異臭がないか確認する。
2. IR入力と7セグ表示を確認する。
3. RGB LEDとブザーの動作を確認する。
4. 最後にサーボの解錠/施錠動作を確認し、リセット発生有無を確認する。

症状別チェック:
- サーボが震える: サーボ赤線/茶線の接続先と電圧降下を確認。
- IRが反応しない: IR端子並び（OUT/GND/VCC）とD2配線を確認。
- 7セグが表示しない: D3/D4逆接続とVCC/GNDを確認。

## 7. ブレッドボード番地指定配線（1〜60）

前提レイアウト: 横は `+-abcdefghij+-`、縦は `1〜60`。

番地の読み方:
- 左電源レール: `L+1`, `L-1` のように表記
- 右電源レール: `R+1`, `R-1` のように表記
- 信号穴: `a1`〜`j60`

### 7-1. 電源系

1. Arduino 5V -> `L+5`
2. Arduino GND -> `L-5`
3. レール橋渡し（左右共通化）
    - `L+1` -> `R+1`
    - `L-1` -> `R-1`

補足:
- `L+1` と `L+5` は同じ +レール上なので電気的に同電位。
- `L-1` と `L-5` は同じ -レール上なので電気的に同電位。
- 同じ穴に2本挿す必要はない（同一レール内の別番地を使ってよい）。

### 7-2. IR受信モジュール（VS1838B）

配置（モジュール本体のピンを以下の番地へ）:
- GND(G):`b8`
- VCC(R): `b9`
- OUT(Y): `b10`

配線:
1. `a8` -> `L-8`（GND行）
2. `a9` -> `L+9`（VCC行）
3. `a10` -> Arduino D2（OUT行）

補足:
- `a8` と `b8`、`a9` と `b9`、`a10` と `b10` はそれぞれ同じ行で同電位。
- モジュール脚は `b` 列、ジャンパは `a` 列にすると同じ穴への2本挿しを避けられる。
- GRY固定の対応は `b8=GND(G)`, `b9=VCC(R)`, `b10=OUT(Y)` とする。

### 7-4. パッシブブザー

配置:
- ブザー+: `e24`
- ブザー-: `f24`（中央溝をまたぐ）

配線:
1. `a24` -> Arduino D8（`a24` と `e24` は同電位）
2. `j24` -> `L-24`（`f24` と `j24` は同電位）

### 7-5. RGB LED（共通カソード）+ 330Ω抵抗3本

LED配置:
- R端子: `e30`
- G端子: `e31`
- B端子: `e32`
- 共通カソード: `e33`

抵抗配置（直列）:
1. R系抵抗: `a30` - `a36`
2. G系抵抗: `a31` - `a37`
3. B系抵抗: `a32` - `a38`

配線:
1. `b36` -> Arduino D10（R抵抗出力行）
2. `b37` -> Arduino D11（G抵抗出力行）
3. `b38` -> Arduino D12（B抵抗出力行）
4. `a33` -> `L-33`（共通カソードをGNDへ）

補足:
- `a36` と `b36`、`a37` と `b37`、`a38` と `b38` は同じ行で同電位。
- 抵抗の足は `a` 列、Arduinoからのジャンパは `b` 列にすると同じ穴への2本挿しを避けられる。

### 7-6. サーボ（SG90）

3ピン受け位置（信号/電源/GND）:
- 信号: `h40`
- 電源(赤): `h41`
- GND(茶): `h42`

配線:
1. `i40` -> Arduino D9（信号行）
2. `i41` -> `L+41`（電源行）
3. `i42` -> `L-42`（GND行）

補足:
- `h40` と `i40`、`h41` と `i41`、`h42` と `i42` は同じ行で同電位。
- サーボの3線は `h` 列、ジャンパは `i` 列に分けると同じ穴への2本挿しを避けられる。

### 7-7. Arduino本体電源

1. 9V1A ACアダプタ -> Arduino DCジャック（またはVin）

### 7-8. 最終確認（番地版）

- `L+` と `L-` の短絡がない
- `a30/a31/a32` の抵抗が正しく `a36/a37/a38` へ直列接続されている
- サーボ赤線は `h41` 系統、茶線は `h42` 系統
- D2/D3/D4/D8/D9/D10/D11/D12 の配線先が重複していない
