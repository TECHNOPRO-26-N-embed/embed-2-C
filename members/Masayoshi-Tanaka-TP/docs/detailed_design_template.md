### `loop()` — メインループ

> ※ loop() は「状態ごとに何をするか」だけ書く。細かい処理は各関数に任せる。

```
【処理の流れ】


＜毎ループ実行すること＞
- readRemote() を呼び出してリモコン入力を確認する

＜fanState が 0（停止）のとき＞
- モータ出力を 0 にする
- LEDを消灯する
- ON/OFFボタンが押されたら fanState = 1

＜fanState が 1（弱）のとき＞
- changeFanSpeed(弱) を実行
- LEDを青色点灯
- 風量変更ボタン → fanState = 2
- ON/OFFボタン → fanState = 0

＜fanState が 2（強）のとき＞
- changeFanSpeed(強) を実行
- LEDを赤色点灯
- 風量変更ボタン → fanState = 1
- ON/OFFボタン → fanState = 0
