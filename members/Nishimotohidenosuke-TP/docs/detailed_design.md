## 2. 各関数の詳細設計

### `setup()` — 初期化処理

```
【処理の流れ】
1. IR受信モジュールのピンモードをINPUTに設定
2. 赤色LEDのピンモードをOUTPUTに設定
3. ブザーのピンモードをOUTPUTに設定
4. IRremoteライブラリの初期化
5. sosCode, cancelCodeに事前計測した値を設定
6. Serial.begin(9600)でデバッグ出力有効化
7. 起動確認としてLEDを1秒点灯
```

---

### `loop()` — メインループ

```
【処理の流れ】

＜毎ループ実行すること＞
- IR信号をreadIR()で受信
- 現在時刻を取得: now = millis()

＜currentState が 0（待機中）のとき＞
- IR信号がsosCodeと一致したら startAlarm() を呼び出し currentState = 1
- showStandby() でLEDをゆっくり点滅

＜currentState が 1（警報発動中）のとき＞
- IR信号がcancelCodeと一致したら stopAlarm() を呼び出し currentState = 0
- updateOutput() でLED高速点滅・ブザーON
```

---

### `readIR()` — 赤外線リモコン信号を受信し、コードを判定

**引数：** なし  
**戻り値：** unsigned long（受信したIRコード）

```
【処理の流れ】
1. IRremoteライブラリで信号受信をチェック
2. 受信があればコード値を返す
3. 受信がなければ0を返す
【エラー・異常ケース】
- 未登録コードの場合は何もしない
```

---

### `updateOutput(state)` — 現在のstateに応じてLED/ブザーを制御

**引数：** state（int）: 現在の状態  
**戻り値：** なし

```
【処理の流れ】
1. state==0ならLEDをゆっくり点滅、ブザーOFF
2. state==1ならLEDを高速点滅、ブザーON
3. millis()を使い周期制御
```

---

### `startAlarm()` — SOSボタン信号受信時に警報（LED点滅・ブザー）を開始

**引数：** なし  
**戻り値：** なし

```
【処理の流れ】
1. ledState, buzzerStateをONに設定
2. 警報用タイマーをリセット
3. updateOutput(1)を呼ぶ
```

---

### `stopAlarm()` — 解除ボタン信号受信時に警報を停止

**引数：** なし  
**戻り値：** なし

```
【処理の流れ】
1. ledState, buzzerStateをOFFに設定
2. updateOutput(0)を呼ぶ
```

---

### `showStandby()` — 待機中にLEDをゆっくり点滅させる

**引数：** なし  
**戻り値：** なし

```
【処理の流れ】
1. millis()で1秒周期ごとにLEDのON/OFFを切り替える
2. 点滅状態をledStateに反映
```

---

## 3. 重要ロジックの詳細設計

### 3-2. millis() を使ったタイマー管理

```
【自分のシステムで millis() を使う処理】
- 待機LED点滅（1000ms周期）
- 警報LED点滅（200ms周期）
- いずれもdelay()は使わず、lastMillisと比較して周期制御
```
